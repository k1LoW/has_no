h1. Simple bind model practice plugin for CakePHP.

This plugin unbind model all, and bind simple.

h2. Features

* unbindModel() all by Model::belongsTo, Model::hasOne, Model::hasMany, Model::hasAndBelongsToMany.
* Provide simple bind model method has(), hasAll().

h2. Usage

First, put `has_no' directory on app/plugins in your CakePHP application.
Then, add the following code in whichever model you want to unbindModel() all (ex. `Post' Model).

<pre>
<?php
class Post extends Model {
    var $actsAs = array('HasNo.HasNo');

    var $hasMany = array(
        'Comment' => array(
            'className' => 'Comment',
            'foreignKey' => 'post_id',
            'dependent' => false,
        )
    );

    var $hasAndBelongsToMany = array(
        'Tag' => array(
            'className' => 'Tag',
            'joinTable' => 'posts_tags',
            'foreignKey' => 'post_id',
            'associationForeignKey' => 'tag_id',
            'unique' => true,
        )
    );

}
</pre>

If you want to use model's bind property, use method @$this->Post->has('AssociationModelName')@ , @$this->Post->has(array('AssociationModelName1', 'AssociationModelName2'))@ or @$this->Post->hasAll()@ .
